# Tillymagic (macOS)
_the time passing terminal game you never knew you needed!_

:warning: flash warning :warning:

![gameplay](mediacontent/sample.gif)

and many more classes, bosses, and maps! the point is _you_ choose your _own_ path.

to play, run:
```
# clone repo
git clone https://github.com/opticalrefraction/tillymagic.git

# open directory
cd tillymagic

# run main script
python3 tillymagic2.py
```

game updates automatically!

like the game? leave a star! :star:
## extra notes
any recommendations or bug reports can go to the issues tab. make sure to search for your issue before creating one. please tag them appropriately when doing so!
