# Tillymagic (macOS)
_old-school combat games turned modern!_

![gameplay](mediacontent/sample.gif)

and many more classes, bosses, and maps!

to play, run:
```
# clone repo
git clone https://github.com/opticalrefraction/tillymagic.git

# open directory
cd tillymagic

# run main script
python3 tillymagic2.py
```
